# Start Frontend

```
# pnpm recommanded
npm i -g pnpm@latest

# install dependencies
pnpm i

# start project (dev mode)
pnpm dev
```
# Backend
**Repo:** https://github.com/Netfloux/netflouxBack

## Step 1 - env
Download the .env file (ask @Robin for link) and place it in /netfloux-backend/.env

## Step 2 - start project

```
# install yarn
npm i -g yarn@latest

# install dependencies
yarn

# start project (dev mode)
yarn develop
```
